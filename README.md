## Technologies

- Express
- Mongoose
- Passport
- OAuth 2.0
- Stripe
- SendGrid
- React
- Redux
- Redux-Form
- React-Stripe-Checkout

### Installation

Install the dependencies

```sh
$ npm run install-all
```

Run app

```sh
$ npm run dev
```
